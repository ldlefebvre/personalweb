EasyFind was my final project for the Le Wagon Web Development Bootcamp — and the
project where the engineering bug really bit.

## The problem

When you move to a new city, getting oriented is overwhelming. EasyFind helps
newcomers quickly find local attractions, events, movies, and restaurants in one
place.

## What I built

A full-stack app that integrates several third-party APIs — PredictHQ for events,
Yelp for restaurants, TMDB for movies, and Mapbox for maps and location — into a
single discovery experience, built on Ruby on Rails with a PostgreSQL database.

## What I took away

EasyFind is where I learned to design around external APIs, model data in a
relational database, and ship a working product under a deadline — the foundation
everything since has been built on.
